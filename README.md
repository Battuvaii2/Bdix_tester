## BDIX_Tester_py
This tool will allow you to test BDIX/FTP server for you ISP


## Prerequisite
<h3>Linux</h3><br>

```bash
pip install wget
pip install requests
sudo apt-get install git
git clone https://github.com/Shantanu2645/BDIX_Tester_py
cd BDIX_Tester_py
python3 bdix.py
```
<br>

<h3>Windows</h3><br>
